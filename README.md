# 💈 Barbershop App

Aplikasi Barbershop berbasis web yang memungkinkan pengguna melakukan booking layanan cukur secara online serta memudahkan admin dalam mengelola data pelanggan dan jadwal.

---

## 📌 Deskripsi

Barbershop App terdiri dari dua bagian utama:
- **Frontend (User & Admin Interface)** menggunakan React
- **Backend (API & Database Management)** menggunakan Node.js dan Express

Aplikasi ini dibuat untuk memberikan pengalaman booking yang mudah, cepat, dan modern.

---

## 🚀 Fitur Utama

### 👤 User
- Booking jadwal cukur
- Memilih barber
- Melihat layanan yang tersedia

### 🛠️ Admin
- Mengelola data barber
- Mengelola jadwal booking

## ⚙️ Instalasi & Menjalankan Project

### 1. Clone Repository
git clone https://github.com/USERNAME/barbershop-app.git
cd barbershop-app

---

### 2. Menjalankan Backend
cd barber-booking-api
npm install
npm run dev

Backend akan berjalan di:
http://localhost:5000

---

### 3. Menjalankan Frontend
cd barber-react
npm install
npm run dev

Frontend akan berjalan di:
http://localhost:5173

---

## 🔗 Koneksi API
Pastikan frontend terhubung ke backend dengan base URL:
http://localhost:5000

---

## 📌 Catatan
- Pastikan Node.js sudah terinstall
